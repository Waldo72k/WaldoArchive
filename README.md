# Como correr este proyecto

Se requiere tener

- Nodejs

utilizar los comandos

> git clone https://github.com/Waldo72k/WaldoArchive

una vez eso, movernos al repositorio con

> cd WaldoArchive

instalar lo necesario con

> npm install

finalmente correrlo con

> npm run dev
